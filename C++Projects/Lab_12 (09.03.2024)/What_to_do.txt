Общее задание
1). Определить класс TMyClass, задав поля согласно своему варианту. Размер статического массива - 5-6 элементов. 
Не забудьте предусмотреть член класса для хранения количества элементов динамического массива. Не рекомендуется 
давать членам класса имена объектов, которые будут созданы в пункте 4 (т.е. а, b, px, ру). Предусмотреть в определении 
класса конструктор по умолчанию, конструктор копии, деструктор, а также функции-члены класса: ввода, вывода и fff 
(в описании класса - только прототипы!) Конструктор по умолчанию должен иметь полный набор параметров со значениями 
по умолчанию для того, чтобы каждое поле инициализировалось осмысленным значением при создании объекта.
2). Прописать реализации конструкторов, деструктора и функций - членов класса. Метод ввода должен давать
возможность вводить с клавиатуры значения для всех полей, включая количество элементов в динамическом массиве). 
Метод вывода должен предусматривать отображение на экране всех полей класса с соответствующими пояснениями 
(имена выводимых полей). ВНИМАНИЕ: реализации не следует помещать в определение класса!
3). Добавить в определение класса прототип внешней дружественной функции ввода значений с клавиатуры. Прописать ее реализацию.
4). Составить программу (функцию main), в которой создать объекты класса. Объекты b создаются как обычные переменные, рх и ру
 - посредством указателей. Обеспечить инициализацию объектов класса способами, указанными в индивидуальном задании. Значения 
полей должны отражать их тип.
5). Обеспечить вывод на экран значений полей всех созданных объектов класса.
6). Вызвать для каждого созданного объекта метод fff.
7). Создать массив объектов класса, используя конструктор по умолчанию, вызвать для каждого элемента метод вывода.
8). Отладить программу. Проанализировать ее работу. Обязательно сохранить: она еще
потребуется.

Индивидуальные данные
Переменная - char
Указатель - unsigned int
Стат. массив - double
Дин. массив - float
Метод класса fff - Умножает все элементы массива на число, переданное как параметр

Конструктор по умолчанию - b
Функция ввода член класса - px
Внешняя функция ввода - py
Конструктор копии (оригинал) - a(b)
