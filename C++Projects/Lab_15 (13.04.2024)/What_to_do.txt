Вариант 20
(общее задание)
1). Создать классы согласно индивидуальному заданию. Помимо необходимых полей он должен содержать конструктор по умолчанию и полным 
набором инициализаторов для всех полей класса, деструктор, функцию ввода с клавиатуры (при необходимости - с проверкой) и функцию 
вывода на экран.
2). Добавить в определение класса перегрузку оператора присваивания (=). Создать и инициализировать несколько объектов класса (число 
объектов и способы инициализации - по мере необходимости). Протестировать перегрузку, проверяя соответствие заданных и выведенных на 
экран данных. Каждое тестирование начинать с вывода всех операндов, после выполнения операции повторять вывод. Подробные пояснения при 
выводе обязательны.
3). Добавить перегрузку операции равно (-) для двух объектов класса. Протестировать перегрузку способом, указанным в п.2. Для
четных вариантов перегружать с помощью друж функции, для нечетных - с помощью метода класса.
4). Добавить перегрузку операции объект+ число (увеличивает на указанное значение все числовые поля, кроме размера динамического массива). 
Для четных вариантов перегружать с помощью метода класса, для нечетных - с помощью друж функции. Протестировать перегрузку способом, 
указанным в п.2. Если индивидуальное задание предполагает агрегирование, этот пункт пропу-
стить.
5-6). Добавить в определение класса перегрузку операций, указанных в индивидуальном задании. Если трактовка самоочевидна либо в задании 
не указана, придумать и обосновать ее самостоятельно. Протестировать способом, описанным в п.2.


(индивидуальное задание)
В этом задании потребуется предварительно разработать класс Кольцо TRing (поля - внешинй и внутренний раднусы). Для него необходимо 
прописать конструкторы, методы и перегрузки для корректного функционирования второго класса. 

Поля второго класса:
Динамический массив Элементов класса Кольцо

5) С помощью друж функции
унарный + (сбрасывает параметры всех колец массива в значения по умолчанию)

6). С помощью метода класса
+ + (увеличивает длину массива на 1 элемент и записывает в него кольцо со значениями по умолчанию)
