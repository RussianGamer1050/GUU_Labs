Вариант 20
1). Разработать класс Река. Поля: название, регион, протяженность участка, пригодного для сплава (в км), 
категория сложности (от 0 до 6), информация по годам за последние пять лет о количестве несчастных случаев 
(статический массив). Методы: ввод значений полей с клавиатуры, вывод значений полей на экран с пояснениями 
и функция, определяющая общее количество несчастных случаев за 5 лет (функция должна возвращать полученное значение).
2). Создать два объекта класса: один с помощью обычной переменной, другой - посредством указателя. Вызвать для
каждого объекта предусмотренные методы.
3). Создать динамический массив из 3-4 объектов. Вызвать для каждого элемента массива предусмотренные методы.
4). Освободить память из-под объекта, созданного при помощи указателя, и из-под массива объектов.
