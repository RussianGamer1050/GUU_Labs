I). Предусмотреть возможность ввода числовых данных как с клавиатуры, так и с помощью датчика 
случайных чисел rand(). Для этого в программе задать диапазон значений a и b для числовых полей, 
который будет использоваться при случайном вводе чисел. Возможность выбора способа ввода 
записей организовать через оператор switch().
II). Для каждой из N введённых записей выполнить задание а) и результат вывести в виде таблицы 
с шапкой.
III). По всем записям по вычисляемому полю таблицы (в задаче о студентах - это Сред.балл) найти 
среднее арифметическое значение этого поля (SR_ALL) и максимум (MAX_ALL) (или минимум)
этого поля.
IV). Организовать вывод записей в виде 3-х таблиц: 
1) таблица записей с данными, у которых вычисляемое поле больше или равно SR_ALL; 
2) таблица записей с данными, у которых вычисляемое поле меньше SR_ALL;
3) таблица записей с данными, у которых вычисляемое поле равно MAX_ALL.
