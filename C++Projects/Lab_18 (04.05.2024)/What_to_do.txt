Для каждого цеха задано: номер, название, Ф.И.О. начальника, укомплектованность в %% по каждой из М специальностей (статический массив), 
план производства на квартал, выполнение плана в % за каждый квартал года (статический массив).
I). Разработвть два класса: класс, характеризующий запись указанного содержания, и класс, характеризующий файл данных, позволяющий 
записывать в файл произвольное количество записей, выводить эту информацию из файла, обрабатывать ее. Классы должны включать необходимые 
конструкторы, перегрузки операций, методы, если надо - деструктор. Методы для работы с файлом данных перечислены в пункте 2.
2). Используя систему ввода и вывода С++ (iostream), разработать программу, позволяющую:
a). Записывать в файл данных информацию указанной в задании структуры,
б). Производить контрольное чтение созданного файла (вывод на экран ВСЕХ записей файла одновременно в виде таблицы; если все они не 
умещаются на экране, предусмотреть задержку после вывода каждых 15-20 записей).
в). Дополнять созданный файл новыми записями (в конец файла).
г). Выдать справку о цехах, задаваемых номерами (количество задаваемых номеров не определено).
3) Организовать меню, обеспечивающее многократно повторять выбранные действия.
