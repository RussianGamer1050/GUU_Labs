Разработать иерархию родственных классов: класс "Канцтовар" - абстрактный базовый класс, от него наследуются классы для товаров: 
1). которыми пишут (TCh1); 2). в которых пишут (TCh2). От класса TChl наследуются классы "Карандаш" и "Ручка". От класса TCh2 
наследуются классы "Тетрадь" и "Альбом". Каждый класс должен содержать конструктор по умолчанию, деструктор (в случае необходимости) 
и не менее трех виртуальных одноименных методов с одинаковыми сигнатурами, например, ввод значений полей с клавиатуры, вывод их на 
экран и обработка - специфический метод, описывающий поведение объекта. Метод обработки должен иметь собственную реализацию для каждого 
класса. Набор полей каждого класса должен быть достаточен для характеристики объекта данного класса. В функции main объявить указатель 
на объект базового класса. Организовать меню, позволяющее многократно выбирать одного из наследников, создавать соответствующий объект 
через указатель на объект базового класса и вызывать для него все виртуальные методы. Использовать возможности динамического полиморфизма.
